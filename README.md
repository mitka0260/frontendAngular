## SETUP:
To start this application you need to:
1) install Node.js from https://nodejs.org/en/download/
2) restart your PC
3) once installed, download this repository (unzip it if needed), go to its directory and run the "npm install" command 
   to install the project dependencies;
4) then run "npm start" to begin the application;
5) after the last command, the application should open in your browser at http://localhost:4200
